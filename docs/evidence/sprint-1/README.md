# Sprint 1 manual evidence records

Use [the record template](manual-test-record-template.md) once per test session. Do
not commit credentials, tokens, parameter values, database dumps, personal data,
device identifiers, public IP addresses, or unrestricted logs.

Expected records before Sprint 1 approval:

1. S01-T01 clean-machine setup and mobile reachability
2. S01-T02 iOS and Android development-build smoke test
3. S01-T07 native secure-storage and leakage audit
4. S01-T08 controlled CI-failure matrix
5. S01-T09 enabled Terraform plan review
6. S01-T10 two-image deployment and rollback
7. S01-T11 alarm, routing, budget, and log-privacy rehearsal
8. S01-T12 disposable restore and credential-rotation review
9. S01-T14 final demonstration and approval

Store small redacted text evidence here. Keep large artifacts in their access-
controlled source and record only an immutable URL, digest, object version, workflow
run, or command ID. Delete field-test resources after rehearsal according to the
cost-control runbook.
