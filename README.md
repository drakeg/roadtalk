# RoadTalk

RoadTalk is a location-aware, voice-first communication platform inspired by classic CB radio culture and designed for modern travelers.

## Project status

RoadTalk is in its planning and repository-initialization phase. Implementation begins only when the applicable sprint is approved and its requirements, acceptance tests, and deliverables are committed here.

## Source of truth

This repository is the authoritative record for:

- project and sprint plans
- requirements and acceptance tests
- sprint deliverables and reviews
- Agile/Kanban working agreements
- architecture decision records
- implementation and validation history

See [Project Governance](docs/PROJECT_GOVERNANCE.md), [Agile Workflow](docs/AGILE_WORKFLOW.md), and the [documentation index](docs/README.md).

## Working rule

Keep changes scoped to the active sprint. Changes to approved sprint scope must be documented and approved before implementation.
