- one iOS/Android codebase with native escape paths
- development builds, not Expo Go as the sole runtime
- platform-secure refresh-credential storage
- no profile, location collection, PTT, proximity, or channel feature during Sprint 1
- physical-device smoke testing when the mobile foundation is implemented
